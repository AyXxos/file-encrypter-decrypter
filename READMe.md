# Encrypter-Decrypter Application

## Description
This application is designed for encryption and decryption analysis and educational purposes.

## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Encrypter-Decrypter
   ```

2. Compile the application:
   ```bash
   gcc -o encrypter-decrypter main.c
   ```

## Usage
Run the application with the following command:
```bash
./encrypter-decrypter [options]
```

### Options
- `-h, --help`: Display help information
- `-v, --version`: Display version information
- ` ` : Open the GUI interface

## Requirements
- GCC compiler
- C standard library
